# Student Admission System

A modern desktop application for managing student records, built with Python (Tkinter) and Google Firebase.

## Features
- **Cloud Database:** Real-time data storage using Firestore.
- **Modern UI:** Clean, dashboard-style interface.
- **CRUD Operations:** Create, Read, Update, and Delete student records.

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install firebase-admin`
3. Add your `serviceAccountKey.json` to the project folder.
4. Run `python Admission_Form.py`.
